   # MyProject

   Этот проект создан в рамках практической работы по созданию структуры проекта и работе с Git.

   ## Структура проекта:
   - `modules/` — содержит модули приложения.
   - `data/` — содержит входные и выходные данные.
   - `docs/` — документация проекта.
